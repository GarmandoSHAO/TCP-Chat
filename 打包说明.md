# 打包说明

用 PyInstaller 将 TCP 聊天室打包成单文件 exe，无需 Python 环境即可运行。

## 打包命令

```bash
cd TCP-Chat

python -m PyInstaller --onefile --noconsole --name "TCP-Chat" \
  --hidden-import tcp_chat.server \
  --hidden-import tcp_chat.tunnel \
  --hidden-import tcp_chat.config \
  --hidden-import tcp_chat.client \
  --distpath . main.py
```

## 参数说明

| 参数 | 作用 |
|------|------|
| `--onefile` | 打包成单个 exe |
| `--noconsole` | 不显示黑色终端窗口 |
| `--name "TCP-Chat"` | exe 文件名 |
| `--hidden-import tcp_chat.*` | 强制包含动态导入的模块 |
| `--distpath .` | exe 输出到当前目录 |

## 打包产物

```
TCP-Chat.exe    约 31MB，自包含，无需 Python
```

## 注意事项

- **bore 隧道**：需要 `bore.exe` 和 `TCP-Chat.exe` 放在同一目录
- **config.json**：可选，不携带则用默认配置
- exe 支持 Windows 7+，直接双击运行
