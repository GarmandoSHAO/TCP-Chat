# 打包说明

一键构建 TCP-Chat 的 **绿色免安装目录** + **Windows 安装包**，以 `TCP-Chat.png` 为图标唯一信源。

## 前置条件

| 依赖 | 版本 | 用途 |
|------|------|------|
| Python | 3.10+ | 运行构建脚本 |
| PyInstaller | 6.x+ | 源码 → EXE |
| Pillow | 10.x+ | PNG → ICO 图标转换 |
| Inno Setup | 6.x | EXE 目录 → 安装包 |

安装命令（一次配好）：

```bash
pip install pyinstaller pillow
# Inno Setup 6.x 需手动安装：https://jrsoftware.org/isinfo.php
```

## 一键打包

```bash
cd TCP-Chat
setup\build_installer.bat
```

脚本自动执行 5 步流水线：

```
Step 1   → 环境检查（Python / PyInstaller / Inno Setup）
Step 1.5 → tools/build_icon.py 从 PNG 重建 ICO
Step 2   → PyInstaller --onedir 打包
Step 3   → 复制外部工具（bore.exe, croc.exe, config.json）
Step 4   → Inno Setup 编译安装包
```

## 打包产物

```
dist/
├── TCP-Chat/                     ← 绿色免安装目录（压缩即可分发）
│   ├── TCP-Chat.exe              ← 主程序（图标已嵌入）
│   ├── config.json
│   ├── bore.exe / croc.exe
│   │── setup_utils.py
│   └── _internal/                ← Python 运行时 + 依赖
│       ├── TCP-Chat.png          ← 窗口图标
│       └── TCP-Chat.ico          ← 多尺寸 ICO
│
└── TCP-Chat-Setup-1.0.0.exe     ← Windows 安装包（推荐分发）
```

## 更换图标

只需替换一个文件：

1. 用你的图标替换 `TCP-Chat.png`（保持同名）
2. 重新运行 `setup\build_installer.bat`
3. 所有环节自动更新：EXE 图标 / 安装包图标 / 窗口图标 / 任务栏图标

## 版本号升级

两个地方需要改：

| 位置 | 字段 |
|------|------|
| `setup/installer.iss` 第 9 行 | `#define MyAppVersion "x.x.x"` |
| `file_version_info.txt` | 三处版本号（`filevers`, `prodvers`, 字符串） |

## 环境检查清单

打包前确认以下文件在项目根目录：

| 文件 | 必须？ | 说明 |
|------|--------|------|
| `TCP-Chat.png` | ✅ 是 | 图标信源 |
| `config.json` | 否 | 默认配置，缺则用内置默认值 |
| `bore.exe` | 否 | 内网隧道，缺少则隧道功能不可用 |
| `croc.exe` | 否 | 文件传输，缺少则文件传输不可用 |

## 分发给用户

| 方式 | 文件 | 用户操作 |
|------|------|----------|
| 安装包（推荐） | `TCP-Chat-Setup-x.x.x.exe` | 双击安装，自动配防火墙 + VC++ 运行库 |
| 绿色版 | `TCP-Chat/` 目录压缩包 | 解压 → 双击 `TCP-Chat.exe` |

## 常见问题

**Q: 安装后桌面快捷方式是 Python 图标？**

A: 已修复。`installer.iss` 中快捷方式显式指定了 `IconFilename`，且 EXE 嵌入了 `RT_VERSION` 资源。如仍有问题，清除 Windows 图标缓存：`ie4uinit.exe -show`。

**Q: 构建报 "Source file config.json does not exist"？**

A: 在项目根目录放一个 `config.json`，或从安装包中复制一份过来。

**Q: 没装 Inno Setup 能打包吗？**

A: 可以。脚本会跳过 Step 4，只产出绿色免安装目录 `dist/TCP-Chat/`。
